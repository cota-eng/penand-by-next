# Bista - アプリ概要

文房具を高速に閲覧し、より正確なレビューを行うことができるレビューサイト。

SEO からの流入をメインとしたいので SSG，ISR を用いた高速なレスポンスを期待する。

SEOの内部対策にも力を入れる。

バックエンドの API は、https://github.com/cota-eng/bista-rest-apiである。

## 実装内容

- Nextjsを用いたフロントエンドの構築
- ログイン(next-auth)
- 検索ページ
- 詳細ページ
- レビュー機能
- 商品のお気に入り登録

## 使用技術

### 言語やフレームワーク

- Next.js(React Hooks)
- tailwindcss
- TypeScript
- Jest

### 外部ツール

- Vercel(Deploy)
- Sentry(導入予定)
